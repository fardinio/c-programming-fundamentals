# C Programming Fundamentals

## Introduction

Welcome to my C Programming Fundamentals repository.

This repository represents the beginning of my programming journey through C Programming, the first programming language I learned. The course was taught by Mahmudur Rahman, Assistant Professor at Daffodil International University, whose teaching style made programming concepts easier to understand and apply in practice. One of the most valuable aspects of the course was the emphasis on logical thinking and problem-solving. Rather than focusing only on writing code.

This repository contains approximately 80 programming problems and their solutions completed during my introductory C Programming course.

The problems cover a wide range of fundamental programming topics and were designed to strengthen logical thinking, analytical skills, and problem-solving abilities. These exercises played an important role in developing my understanding of programming fundamentals and preparing me for more advanced programming concepts.

### Topics Covered

* Basic Input and Output
* Variables and Data Types
* Operators and Expressions
* Conditional Statements
* Loops and Iteration
* Functions
* Arrays
* Strings
* Pattern Printing
* Mathematical Problems
* Basic Problem Solving Techniques
